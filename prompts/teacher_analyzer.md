# 教师风格分析模板

## 分析目标

从教学材料中提取教师的个人风格，构建5层人格结构：
1. Layer 0：教育理念（核心原则）
2. Layer 1：教师身份（职业认知）
3. Layer 2：沟通风格（表达方式）
4. Layer 3：课堂决策（行为模式）
5. Layer 4：师生关系（人际互动）

## 输入数据

### 数据来源
- 课堂语言记录（文字稿）
- 教学反思日记
- 与学生沟通记录
- 教研活动发言
- 公开课评课记录
- 社交媒体发言（如有）

### 重点关注
1. **语言特征**：口头禅、高频词、句式特点
2. **情感表达**：语气、情绪、态度
3. **互动模式**：提问方式、回应风格、反馈习惯
4. **决策逻辑**：问题处理、冲突解决、方案选择

## 5层分析框架

### Layer 0：教育理念（不可违背的核心原则）
**分析内容**：
- 根本教育信念
- 教学底线原则
- 价值排序（知识vs能力vs情感）
- 不可妥协的规则

**提取方法**：
1. 寻找重复出现的价值观表述
2. 识别"必须"、"绝不"等绝对化表达
3. 分析决策背后的原则性理由

**示例输出**：
```yaml
layer_0:
  core_beliefs:
    - "教育是唤醒，不是灌输"
    - "每个学生都有闪光点"
    - "错误是学习的最好机会"
  
  non_negotiables:
    - "绝不公开批评学生"
    - "必须给每个学生发言机会"
    - "作业反馈不超过24小时"
  
  value_hierarchy:
    primary: "学生成长"
    secondary: "知识掌握"
    tertiary: "考试成绩"
```

### Layer 1：教师身份（职业自我认知）
**分析内容**：
- 角色定位（引导者、权威、朋友等）
- 专业认同感
- 职业使命感
- 自我效能感

**身份标签**：
- 知识传授者
- 学习引导者
- 成长陪伴者
- 人生导师
- 严师慈母/父

**提取方法**：
1. 分析第一人称表述（"我作为老师..."）
2. 识别角色隐喻（"像园丁一样..."）
3. 分析责任归属表述

**示例输出**：
```yaml
layer_1:
  primary_role: "学习引导者"
  secondary_roles: ["成长陪伴者", "知识传授者"]
  
  professional_identity:
    strength: "学科专家"
    mission: "激发学习兴趣"
    challenge: "平衡个性与统一"
  
  self_efficacy:
    confidence: 0.8
    growth_mindset: true
    adaptability: 0.75
```

### Layer 2：沟通风格（语言表达特征）
**分析内容**：
- 语言正式程度
- 情感温度
- 幽默感使用
- 鼓励方式
- 批评艺术

**风格维度**：
1. **正式-亲切**：语言规范程度
2. **直接-委婉**：意见表达方式
3. **严肃-幽默**：情绪基调
4. **简洁-详细**：信息密度

**提取方法**：
1. 词频分析（口头禅、高频词）
2. 句式分析（疑问句、感叹句、祈使句比例）
3. 修辞分析（比喻、夸张、反问等）
4. 情感分析（积极/消极词汇比例）

**示例输出**：
```yaml
layer_2:
  verbal_habits:
    catchphrases: ["我们来思考一下", "这个想法很有意思", "还有没有其他可能"]
    high_frequency: ["理解", "思考", "尝试", "分享"]
    sentence_patterns: ["如果...那么...", "不仅...而且...", "为什么...因为..."]
  
  communication_style:
    formality: 0.6  # 0-1，0最随意，1最正式
    directness: 0.7
    humor_level: 0.4
    warmth: 0.8
  
  feedback_patterns:
    praise: ["具体肯定+原因", "公开表扬"]
    correction: ["私下指出+建议", "先肯定后改进"]
    encouragement: ["强调进步", "降低焦虑"]
```

### Layer 3：课堂决策（行为模式）
**分析内容**：
- 问题处理流程
- 冲突解决策略
- 突发情况应对
- 时间管理习惯
- 资源分配原则

**决策模式**：
- **分析型**：数据驱动，逻辑严密
- **直觉型**：经验导向，快速反应
- **民主型**：集体决策，征求意见
- **权威型**：独自决定，明确指令

**提取方法**：
1. 分析决策案例记录
2. 识别决策触发条件
3. 归纳决策评估标准
4. 总结决策执行方式

**示例输出**：
```yaml
layer_3:
  decision_making:
    primary_mode: "分析型"
    secondary_mode: "民主型"
    
    process: ["收集信息", "分析选项", "征求意见", "做出决定", "评估效果"]
    criteria: ["学生受益", "教育价值", "可行性", "公平性"]
    speed: "中等"  # 快速/中等/谨慎
    
  problem_solving:
    approach: "根本原因分析"
    steps: ["描述问题", "分析原因", "生成方案", "选择实施", "跟踪效果"]
    resources: ["同事经验", "教育理论", "学生反馈"]
    
  time_management:
    planning: "详细计划+弹性调整"
    prioritization: "重要>紧急"
    delegation: "适度授权学生"
```

### Layer 4：师生关系（人际互动）
**分析内容**：
- 距离把握（亲密-疏远）
- 边界设置（严格-宽松）
- 信任建立方式
- 冲突处理模式
- 情感支持提供

**关系类型**：
- **权威型**：明确边界，强调尊重
- **民主型**：平等对话，共同决策
- **朋友型**：情感亲近，界限模糊
- **导师型**：专业指导，成长关注

**提取方法**：
1. 分析互动记录中的权力动态
2. 识别关系建立的关键事件
3. 归纳信任建立的模式
4. 总结冲突处理的风格

**示例输出**：
```yaml
layer_4:
  relationship_style:
    primary: "导师型"
    secondary: "民主型"
    
    distance: "适度亲近"  # 亲密/适度/疏远
    boundaries: "清晰但灵活"
    trust_building: ["兑现承诺", "保守秘密", "公平对待"]
    
  emotional_support:
    availability: "课间和课后"
    approach: "主动观察+适时介入"
    methods: ["倾听", "共情", "建议", "转介"]
    
  conflict_management:
    prevention: ["明确规则", "及时沟通", "公平处理"]
    resolution: ["私下谈话", "双方陈述", "共同解决", "后续跟进"]
    learning: ["转化为教育机会"]
```

## 综合分析

### 风格一致性检查
检查各层之间的一致性：
- Layer 0理念是否贯穿所有行为
- Layer 1身份是否与沟通风格匹配
- Layer 3决策是否体现Layer 4关系

### 矛盾点识别
识别可能存在的矛盾：
- 理念宣称 vs 实际行为
- 不同情境下的风格差异
- 理想自我 vs 现实表现

### 发展轨迹分析
如果有多时间点数据：
- 风格演变趋势
- 关键转折事件
- 成熟度变化

```yaml
    # ... 具体内容

  layer_4:  # 师生关系

```

### 自然语言摘要
```
张老师的教育理念核心是"以学生为中心"，坚信每个学生都能成功。
他认同自己作为"学习引导者"的角色，专业自信但保持谦逊。
沟通风格亲切而不失严谨，善于用提问引导学生思考。
决策时注重分析和民主，会充分考虑学生意见。
师生关系把握适度距离，既是严师也是成长伙伴。
```

## 应用指导

### 生成teacher.md
基于分析结果生成人格文档：
- 每层详细描述
- 具体行为示例
- 情境应对指南
- 成长建议

### 与教学能力整合
将教师风格与教学能力关联：
- 沟通风格影响教学方法选择
- 决策模式影响课堂管理
- 师生关系影响评价方式
- 教育理念影响目标设定
