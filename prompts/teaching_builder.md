# 教学能力文档生成模板

## 生成目标

基于教学能力分析结果，生成结构化的 `teaching.md` 文档，包含：
1. 学科知识体系
2. 教学方法库
3. 备课流程
4. 评价策略
5. 资源建设
6. 典型教学案例
7. 专业发展

## 输入数据

### 必需数据
- 教师基本信息（姓名、学科、教龄等）
- 教学能力分析结果（来自 `teaching_analyzer.md` 输出）
- 教学材料分析摘要

### 可选数据
- 学生评价摘要
- 同行评价
- 教学成果记录

## 生成规则

### 内容填充规则
1. **基于证据**：所有描述必须有分析结果支持
2. **具体明确**：避免空泛描述，提供具体细节
3. **结构清晰**：层次分明，逻辑连贯
4. **语言专业**：使用教育专业术语，但不晦涩

### 优先级规则
1. **高频特征优先**：出现频率高的特征重点描述
2. **独特特征突出**：与众不同的特点单独强调
3. **有效特征优先**：实践证明有效的做法重点介绍

### 语言风格
- **正式程度**：专业但不刻板（正式度：7/10）
- **人称使用**：第三人称描述
- **时态**：一般现在时（描述常态）
- **语气**：客观陈述，适度评价

## 质量检查

### 完整性检查
- [ ] 是否覆盖所有核心维度
- [ ] 是否有具体案例支撑
- [ ] 是否有数据或证据支持
- [ ] 是否体现教师特色

### 一致性检查
- [ ] 各部分描述是否一致
- [ ] 是否与教师风格协调
- [ ] 是否与实际情况相符

### 实用性检查
- [ ] 是否对Skill运行有指导价值
- [ ] 是否包含可操作的方法
- [ ] 是否有助于理解教师特点

### 输出文档示例
```markdown
# 张老师 - 教学能力

## 学科知识体系

### 数学知识结构
- **核心领域**：函数与导数、解析几何、概率统计
- **知识关联**：注重不同章节间的内在联系，如函数思想在几何中的应用
- **重点难点**：导数应用、空间向量、条件概率

### 知识讲解特点
1. **生活化引入**：用摩天轮问题引入三角函数，用彩票中奖讲解概率
2. **可视化辅助**：善用几何画板、动态演示
3. **思维导图**：每章结束绘制知识网络图

## 教学方法库

### 主要教学方法
1. **启发式教学**
   - 问题链设计：由浅入深的问题序列
   - 探究活动：让学生自己发现规律
   - 错误分析：将典型错误转化为学习资源

2. **差异化教学**
   - 分层作业：A组（基础）、B组（提高）、C组（拓展）
   - 小组合作：异质分组，优势互补
   - 个别辅导：课后针对性指导

# ... 后续内容
```

## 特殊处理

### 数据不足情况
如果某些维度数据不足：
1. **明确标注**："基于现有材料，{维度}信息有限"
2. **合理推断**：基于已知信息合理推断
3. **留出空间**：结构保留，内容简略

### 矛盾信息处理
如果分析结果存在矛盾：
1. **识别矛盾**：明确矛盾点
2. **分析原因**：可能是情境差异或发展阶段不同
3. **辩证描述**："在某些情况下...，在另一些情况下..."

### 发展阶段描述
根据教龄描述发展阶段：
- **新手教师**（1-3年）：侧重基本功、模仿学习
- **熟练教师**（4-10年）：侧重风格形成、方法创新
- **骨干教师**（11-20年）：侧重经验提炼、引领示范
- **专家教师**（20年以上）：侧重思想形成、培养新人
