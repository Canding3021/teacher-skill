# 教师信息录入模板

## 问题序列

请按顺序询问以下问题，每个问题后等待用户回答。

### 问题 1：教师基本信息
**提示语**：请输入教师的姓名或代号（必填）：

**收集字段**：
- `name`：姓名/代号
- `slug`：自动生成（姓名拼音或自定义）

**处理规则**：
- 如果用户输入中文姓名，自动生成拼音slug（如"张三" → "zhangsan"）
- 如果用户输入英文或代号，直接使用
- 确认slug唯一性

### 问题 2：职业信息
**提示语**：请描述教师的基本信息（学科、教龄、学校、职称，想到什么写什么）：

示例：`高中数学，15年教龄，市一中，高级教师`

**收集字段**：
- `subject`：学科
- `years`：教龄
- `school`：学校
- `title`：职称
- `grade`：年级（如"高一"、"高三"）

**处理规则**：
- 从描述中提取关键信息
- 教龄转换为数字（如"15年" → 15）
- 学科标准化（如"数学" → "数学"，"math" → "数学"）

### 问题 3：教学风格
**提示语**：请描述教师的教学风格（教学理念、课堂风格、特色方法）：

示例：`启发式教学，严谨型，擅长用生活实例引入概念，注重思维训练`

**收集字段**：
- `philosophy`：教育理念
- `style`：课堂风格
- `methods`：教学方法
- `specialties`：教学特色

**标签提取**：
从描述中提取标准化标签：
- 教学风格：严谨型、活泼型、民主型、权威型
- 教学方法：启发式、探究式、合作学习、翻转课堂、差异化教学
- 课堂管理：严格管理、自主管理、情感引导

### 问题 4：代表性成果
**提示语**：请列出教师的代表性成果（获奖、课题、公开课、论文等）：

示例：`省级优质课一等奖，主持市级课题《数学建模教学研究》，发表论文3篇`

**收集字段**：
- `awards`：获奖情况
- `projects`：研究课题
- `open_classes`：公开课
- `papers`：论文发表
- `other_achievements`：其他成果

## 信息汇总与确认

收集完信息后，向用户展示汇总：

```
教师信息汇总：

1. 基本信息
   姓名：{name}（{slug}）
   学科：{subject}，教龄：{years}年
   学校：{school}，职称：{title}

2. 教学风格
   教育理念：{philosophy}
   课堂风格：{style}
   特色方法：{methods}
   标签：{tags}

3. 代表性成果
   {achievements_summary}

确认信息正确？如需修改请告诉我。
```

## 特殊处理

### 跳过处理
如果用户对某个问题回答"跳过"或留空：
- 记录为"未提供"
- 在汇总中标记为"待补充"
- 不影响流程继续

### 多教师处理
如果用户一次提供多个教师信息：
- 分别记录
- 生成不同的slug
- 分别确认

### 信息修正
如果用户说"修改"或"不对"：
- 询问要修改的部分
- 重新收集该部分信息
- 更新汇总
